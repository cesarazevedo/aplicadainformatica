Algoritmos, Lógica e Estruturas de Controle

O que são Algoritmos?

Um algoritmo é uma sequência de instruções realizadas com o objetivo de solucionar um problema. Em programação, utilizamos algoritmos para descrever as etapas que precisam ser efetuadas para que um programa execute as tarefas que lhe são designadas.

Características dos Algoritmos:

  • Conjunto de passos finitos e organizados
  • Quando executados, resolvem um determinado problema
  • São compostos por entrada (input), processamento e saída (output)

Pseudocódigo (Portugol)

O pseudocódigo é um método genérico de representar um algoritmo, usando uma linguagem simples sem necessidade de conhecer a sintaxe de linguagem de programação específica.

algoritmo "exemplo_basico" var numero: inteiro dobro: inteiro inicio leia(numero) dobro ← numero * 2 escreva("O dobro é: ", dobro) fim

Variáveis e Tipos de Dados

Variáveis são espaços alocados na memória do computador para armazenar dados ou expressões utilizados por um algoritmo.

Tipos de Dados Principais:

  • inteiro: Números inteiros (10, -5, 42)
  • real: Números decimais (3.14, -2.5)
  • logico: Valores booleanos (verdadeiro, falso)
  • caractere: Sequência de caracteres ("Olá, mundo!")

Regras para nomes de variáveis:

  • Devem ser significativos
  • Usar padrão snake_case (palavras_separadas_por_underscore)
  • Começar com letra ou underscore
  • Não podem ser palavras reservadas

Estruturas de Controle

As estruturas de controle determinam o fluxo de execução de um algoritmo, permitindo decidir a ordem dos comandos.

1. Estrutura Sequencial

Executa comandos em ordem, um após o outro.

leia(nome) escreva("Olá, ", nome)

2. Estrutura Condicional Simples

Executa comandos apenas se a condição for verdadeira.

se numero > 0 então escreva("Número positivo") fimse

3. Estrutura Condicional Composta

Permite avaliar múltiplas condições.

se numero > 0 então escreva("Positivo") senão se numero < 0 então escreva("Negativo") senão escreva("Zero") fimse

Exercícios Práticos

Exercício 1: Calculadora de Idade

Escreva um algoritmo em pseudocódigo que leia o ano de nascimento de uma pessoa e calcule e imprima a idade dela (considere o ano atual como 2025).

Exercício 2: Verificação de Número Par

Desenvolva um algoritmo que leia um número inteiro e verifique se ele é par. Se for par, exiba "O número é par", caso contrário, não exiba nada.

Exercício 3: Classificação de Nota

Crie um algoritmo que leia uma nota (0 a 10) e classifique conforme os critérios:

  • Nota >= 9: "Excelente"
  • Nota >= 7: "Bom"
  • Nota >= 5: "Regular"
  • Nota < 5: "Insuficiente"

Exercício 4: Calculadora de IMC

Desenvolva um algoritmo que calcule o IMC (peso / altura²) e classifique o resultado:

  • IMC < 18.5: "Abaixo do peso"
  • IMC 18.5-24.9: "Peso normal"
  • IMC 25-29.9: "Sobrepeso"
  • IMC >= 30: "Obesidade"
